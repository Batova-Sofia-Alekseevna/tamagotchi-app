# tamagotchi-app
Это проект игры, где пользователь ухаживает за виртуальным питомцем. Главная цель — вовремя закрывать потребности существа: кормить, лечить и развлекать. Состояние питомца (голод, здоровье, настроение) меняется в реальном времени и зависит от качества заботы. Это простая симуляция жизни, обучающая ответственности через игровой процесс.
